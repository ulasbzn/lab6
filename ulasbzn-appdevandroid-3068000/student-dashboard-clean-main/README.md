# Student Dashboard

React + Vite lab project for managing a simple student list.

## Getting Started

```bash
npm install
npm run dev
```

This starts the Vite development server (typically at [http://localhost:5173](http://localhost:5173)).

## Project Structure

- `src/App.jsx` – top-level application logic and state
- `src/components/` – form, list, controls, and item components
- `src/styles/lab-styles.css` – provided lab stylesheet

## Notes

- `node_modules/` and build outputs are ignored via `.gitignore`.
- Run `npm run build` if you need a production bundle.
